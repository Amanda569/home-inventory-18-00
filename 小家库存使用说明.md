# 小家库存使用说明

## 1. 这个网页是干什么的

小家库存的目标很简单：

1. 记录家里有哪些东西。
2. 记录这些东西放在什么位置。
3. 记录这些位置属于哪个区域。
4. 及时看出哪些东西快用完了，方便补货。

你可以把它理解成一个“手机上就能用的家庭库存台账”。

它适合记录的内容包括：

- 纸巾、牙膏、洗发水、洗衣液等日用品
- 螺丝刀、胶带、充电线等工具
- 零食、药品、备用电池等杂项
- 任何需要记住“放哪了、还剩多少”的东西

## 2. 先认识页面上有哪些入口

打开网页后，最主要的入口有这些：

1. 底部导航栏
   - `物品`
   - `位置`
   - `区域`
   - `补货`
   - `设置`

2. 右下角 `+`
   - 用来快速新增物品

3. 顶部搜索框
   - 用来搜索物品、位置、区域关键词

4. 右上角同步按钮
   - 用来手动同步数据

## 3. 先搞清楚“物品、位置、区域”的区别

这个非常重要，后面所有操作都靠这个逻辑。

### 3.1 物品

物品是“具体的东西”，比如：

- 牙膏
- 纸巾
- 螺丝刀套装
- 备用电池

物品记录的是：

- 名称
- 分类
- 存放位置
- 数量
- 单位
- 是否需要低库存提醒
- 备注、品牌、规格、购买日期、到期日期、缩略图

### 3.2 位置

位置是“更具体的存放点”，比如：

- 床下-A箱
- 衣柜-上层
- 桌面-左抽屉
- 阳台-收纳架2层
- 卫生间-镜柜

位置用于回答“这个东西具体放在哪个角落里”。

### 3.3 区域

区域是“更大的分类范围”，比如：

- 床下
- 衣柜
- 桌面
- 阳台
- 卫生间
- 厨房
- 临时
- 其他

你可以理解成：

- 区域是大类
- 位置是大类下面的具体格子

例如：

- 区域：床下
- 位置：床下-A箱、床下-B箱

## 4. 最常用的操作顺序

建议平时按这个顺序使用：

1. 先看 `物品`
2. 再看 `位置`
3. 再看 `区域`
4. 快没了就去 `补货`
5. 需要改同步、导出、清空等设置时再去 `设置`

如果你只是想“赶紧记一件东西放哪了”，最简单的方法就是：

1. 点右下角 `+`
2. 填物品名
3. 选分类
4. 选位置
5. 填数量
6. 点保存

## 5. 如何新增一个物品

这是最常用的功能。

### 方法一：直接点右下角 `+`

适合新增任意物品。

操作步骤：

1. 打开任意页面。
2. 点右下角 `+`。
3. 弹出“新增物品”窗口。
4. 填写物品名。
5. 选择分类。
6. 选择位置。
7. 填数量和单位。
8. 如果需要补货提醒，勾选提醒并设置最低数量。
9. 点 `保存`。

### 方法二：从某个位置里直接新增

适合你已经知道东西要放在哪。

操作步骤：

1. 进入 `位置` 页。
2. 点某个位置卡片，打开这个位置的详情。
3. 点 `添加物品`。
4. 新物品会自动绑定到当前这个位置。
5. 只要补上物品名、分类、数量等信息就行。

这比先去找位置、再手动选位置更快。

## 6. 新增物品时每个字段怎么填

### 6.1 物品名

这是必填项。

建议写法：

- 牙膏
- 纸巾
- 备用电池
- 螺丝刀套装

尽量写成一看就懂的名字，不要写得太口语化。

### 6.2 分类

分类用于快速筛选。

常见分类可以是：

- 日用
- 洗漱
- 工具
- 电子
- 食品
- 药品
- 文具
- 其他

如果下拉里没有你想要的分类，可以直接输入或选择最接近的一类。

### 6.3 位置

这里填具体存放点。

推荐写法：

- 床下-A箱
- 衣柜-上层
- 桌面-左抽屉
- 卫生间-镜柜

不要只写“床下”这种太宽的名字，尽量写到箱、层、抽屉这种级别。

### 6.4 数量 / 余量

这里填当前剩余量。

例子：

- 牙膏还剩 1 支，就填 `1`
- 纸巾还剩 3 包，就填 `3`
- 0.5 这种小数也支持

### 6.5 单位

用于显示更直观。

例子：

- 支
- 包
- 瓶
- 个
- 套
- 节

填写后，页面会显示成类似：

- `1支`
- `3包`

### 6.6 需要低库存提醒

如果勾上这个选项，系统会帮你盯着剩余量。

建议这些东西打开提醒：

- 纸巾
- 牙膏
- 洗衣液
- 电池
- 常用调味料

### 6.7 低于或等于

这是“提醒线”。

比如：

- 设为 `1`
- 那么数量到 `1` 或更少时，就会进入补货提醒

如果某个东西不想提醒，就不要勾选提醒。

### 6.8 备注

适合写一些额外说明。

例子：

- 5号电池
- 备用装
- 只剩半瓶
- 给室友共用

### 6.9 更多资料

点开“更多资料”后，可以补充：

- 品牌
- 规格
- 购买日期
- 到期日期
- 缩略图链接

这些不是每次都要填，只有你想记得更细时再补。

## 7. 如何查看一个物品的完整信息

操作步骤：

1. 在 `物品` 页找到目标物品。
2. 点这个物品卡片。
3. 会打开“物品详情”窗口。

这里会看到：

- 物品缩略图
- 分类
- 数量
- 存放位置
- 备注
- 品牌
- 规格
- 购买日期
- 到期日期
- 是否开启低库存提醒
- 更新时间

详情页右下方通常有两个按钮：

- `看位置`
- `编辑`

含义：

- `看位置`：直接跳到这个物品所在的位置
- `编辑`：修改这个物品信息

## 8. 如何修改一个物品

### 方式一：从物品详情里改

步骤：

1. 点物品卡片。
2. 打开详情。
3. 点 `编辑`。
4. 修改需要调整的字段。
5. 点 `保存`。

### 方式二：直接在列表中找到再改

如果你已经知道它在哪里，也可以先在列表里找，再进详情编辑。

### 修改后会不会丢

不会。

保存后会同步到当前设备，并在开启云同步时同步到云端。

## 9. 如何删除一个物品

操作步骤：

1. 打开这个物品的详情。
2. 点 `编辑`。
3. 在编辑窗口里点 `删除`。
4. 确认删除。

删除后，这个物品就不会再出现在列表里。

## 10. 如何快速调整数量

有些场景不需要重新编辑整条记录，只是想简单加减数量。

常见用法：

- 拿走一包纸巾，数量减 1
- 新买了两瓶洗衣液，数量加 2

如果页面里有加减按钮，直接点即可。

如果没有看到，就用 `编辑` 改数量。

## 11. `物品` 页怎么用

`物品` 页是你最常看的清单页。

这里主要有三个作用：

1. 看全部物品
2. 按分类筛选
3. 搜索物品

### 11.1 分类筛选

在物品列表上方，会有分类筛选按钮。

点某个分类后，页面只显示这个分类的物品。

比如你点：

- `洗漱`

页面就只看见牙膏、洗发水、洗面奶这一类内容。

### 11.2 搜索

顶部搜索框可以搜：

- 物品名
- 分类
- 位置
- 备注
- 品牌
- 规格
- 购买日期
- 到期日期

比如你输入：

- `牙膏`
- `床下`
- `5号电池`

都会自动匹配。

搜索结果会显示在搜索框下面，点结果就能直接打开物品详情。

### 11.3 示例导入

如果当前列表还很空，页面里可能有 `导入示例` 按钮。

它的作用是：

- 快速塞一些样例数据
- 帮你理解这个网页怎么用

如果你已经开始正式记账了，这个按钮就不一定要用。

## 12. `位置` 页怎么用

`位置` 页是最适合找“东西放哪了”的页面。

每个位置卡片通常会显示：

- 位置名称
- 所属区域
- 这个位置里有多少物品
- 是否有待补物品
- 说明文字

### 12.1 点击位置卡片

点卡片本体，会进入这个位置的详情。

这里会看到这个位置下所有物品。

### 12.2 点右上角设置按钮

每个位置卡片右上角都有一个设置按钮。

点它会弹出一个小菜单，通常只有：

- `编辑`
- `删除`

注意：

- 点卡片本体是“查看”
- 点右上角设置按钮是“操作菜单”

这两种动作是分开的。

### 12.3 位置详情里能做什么

进入某个位置后，你会看到：

- 这个位置里的全部物品
- 网格视图
- 表格视图
- 物品缩略图
- `添加物品`

#### 网格视图

适合手机快速扫一眼。

#### 表格视图

适合想看字段更完整的时候。

#### `添加物品`

这是很实用的按钮。

点它后，新建物品会自动绑定到当前位置。

也就是说：

- 你不用再重新选位置
- 直接补物品名和数量就行

### 12.4 修改位置

进入位置编辑后，可以改：

- 位置名称
- 所属区域
- 说明

比如你把：

- `床下-A箱`

改成：

- `床下-左侧收纳箱`

系统会把原先这个位置下的物品位置一起更新掉。

### 12.5 删除位置

删除前要注意：

- 如果这个位置下面还有物品，系统通常不允许直接删
- 你要先把物品挪走，或者先改到别的位置

这样可以避免物品“找不到家”。

## 13. `区域` 页怎么用

`区域` 页是更大的分类页。

它适合先看一个大范围，再从里面找具体位置。

### 13.1 点击区域卡片

点区域本体，会打开这个区域的详情。

里面会列出这个区域下的所有位置。

### 13.2 点右上角设置按钮

区域卡片右上角也有设置按钮。

通常只提供：

- `编辑`
- `删除`

### 13.3 区域详情里能做什么

进入区域详情后，可以：

- 看该区域下有哪些位置
- 点某个位置直接进入它的物品视图
- 点 `添加位置`

### 13.4 `添加位置`

这是区域页里最有用的功能之一。

它的意思是：

- 在当前这个区域下面新建一个位置

比如你现在在 `床下` 区域里，点 `添加位置`，新建的这个位置会自动归到 `床下`。

### 13.5 修改区域

如果把区域名改掉，比如：

- `卫生间`

改成：

- `洗漱区`

那原本属于这个区域的位置会一起跟着更新。

### 13.6 删除区域

区域不能随便删。

如果这个区域下还有位置，系统会先提醒你把位置移走，再删除区域。

## 14. `补货` 页怎么用

`补货` 页专门用来看快用完的东西。

进入后会显示：

- 需要补货的物品列表
- 还剩多少
- 在哪里

哪些东西会出现在这里？

- 勾选了 `需要低库存提醒`
- 且当前数量已经小于或等于提醒线

例子：

- 牙膏还剩 1 支，提醒线是 1
- 那它就会出现在补货页

这个页面的意义是：

- 你不用每次翻全部物品
- 只要打开补货页，就知道该买什么了

## 15. `设置` 页怎么用

`设置` 页主要管三件事：

1. 家庭名称
2. 成员名称
3. 数据同步和导入导出

### 15.1 家庭名称

这个名字主要用于显示整个空间的名称。

比如：

- 我们的小家
- 合租房库存
- 1号房间仓库

### 15.2 成员名称

这个是给当前操作的人看的。

比如：

- 我
- 室友A
- 室友B

这样以后看更新时间时更容易知道是谁改的。

### 15.3 同步状态

设置页里会显示当前同步状态。

可能看到的状态有：

- 本机模式
- 云同步已启用
- 同步失败

如果是本机模式，说明数据暂时只存在当前设备浏览器里。

如果启用了云同步，手机和电脑可以看到同一份数据。

### 15.4 导出

导出会生成一个 JSON 文件。

用途：

- 备份
- 迁移
- 出问题时恢复数据

建议你隔一段时间导出一次，放到网盘里。

### 15.5 导入

导入适合拿回备份数据。

注意：

- 导入的是 JSON 文件
- 导入后会覆盖当前设备上的数据结构

所以导入前最好先导出一份备份。

### 15.6 清空

清空会把当前数据全部清掉。

这个按钮只建议在你明确知道自己在做什么的时候使用。

## 16. 两个人一起用时怎么配合

如果你和室友都要用，建议这样约定：

### 16.1 统一一个总规则

建议先约定：

- 谁负责买东西
- 谁负责改数量
- 谁负责整理位置

### 16.2 同一条数据尽量别同时改

如果两个人同时编辑同一个物品，后保存的人可能覆盖前一个人的修改。

最稳妥的方式是：

- 一次只改一个人手上的那条
- 改完就保存

### 16.3 看到变化后先刷新一下

如果你刚改完，室友还没看到更新，可以：

1. 等几秒
2. 刷新页面
3. 点右上角同步按钮

### 16.4 以“云同步”作为共同数据源

两个人最好都打开同一个网页地址，并且都连着网络。

这样你们看到的会是同一份共享库存。

## 17. 手机上怎么用最顺手

手机上建议这样操作：

1. 优先用底部导航切换页面
2. 想新增东西时直接点右下角 `+`
3. 想找东西时先用搜索框
4. 想查某个位置时直接进 `位置`
5. 想看快没的东西时进 `补货`

手机屏幕小，最省事的做法是：

- 先搜
- 再点结果
- 再看详情
- 再编辑

## 18. 常见使用习惯建议

### 18.1 新买东西时就顺手记

不要等到东西全混乱了才补录。

最好的习惯是：

- 拿到快递
- 放好位置
- 立刻记一条

### 18.2 位置名字尽量固定

比如固定成：

- 床下-A箱
- 床下-B箱
- 衣柜-上层
- 阳台-收纳架2层

这样以后搜索更快，不容易乱。

### 18.3 大件和小件分开记

比如：

- 纸巾一条一个位置
- 工具箱一个位置
- 电池一个位置

不要把很多东西都塞成一个模糊位置，不然以后会不好找。

### 18.4 低库存提醒只给“会反复买”的东西

不是所有东西都要提醒。

最适合开提醒的是：

- 纸巾
- 牙膏
- 洗衣液
- 电池

## 19. 如果出现问题，先看这里

### 19.1 找不到刚刚新增的内容

先检查：

1. 有没有点 `保存`
2. 有没有切到正确页面
3. 有没有刷新
4. 有没有同步成功

### 19.2 电脑能看到，手机看不到

通常是这几个原因：

1. 手机没有打开同一个网页地址
2. 其中一台设备还没同步成功
3. 网络断开
4. 配置没有指向同一个云端空间

### 19.3 删除后又回来了

一般说明：

- 还没有真正保存成功
- 或者另一台设备又把旧数据同步回来了

建议：

1. 先确认是否已经同步成功
2. 再刷新页面
3. 再看列表

### 19.4 修改名称后出现重复

正常情况下，改位置名或区域名会自动联动原数据。

如果你看到重复，建议：

1. 刷新一下页面
2. 再看是否还存在
3. 如果仍重复，检查是否是另一个旧位置名残留

### 19.5 补货页没有内容

这说明：

- 当前没有开启提醒的物品
- 或者所有开启提醒的物品还没到提醒线

### 19.6 页面很空

如果一开始没数据，可以先点：

- `导入示例`

这样就能快速看到一个可用的样子。

## 20. 最后给室友的简化版口诀

如果她只记得住一句话，可以记成：

1. `物品` 里看东西本身。
2. `位置` 里看东西放哪。
3. `区域` 里看大范围位置。
4. `补货` 里看快没的东西。
5. 右下角 `+` 是最快的新增入口。
6. 位置页点卡片看详情，点右上角设置做编辑或删除。
7. 区域页点卡片看里面有哪些位置，点设置只做编辑或删除。
8. 新增物品时，先填名字，再选位置，最后点保存。

 
## 21. 最新录入规则

现在新增物品时，录入顺序变成这样：

1. 分类可以直接输入，也可以从推荐项里选。
2. 先选或输入区域。
3. 再选或输入位置。
4. 如果区域是新名字，保存后会自动加入“区域”模块。
5. 如果位置是新名字，保存后会自动加入“位置”模块，并自动绑定到刚才选的区域。

这样做的好处是：

- 不用先去“位置”页手动补登记
- 新区域、新位置、物品记录可以一次完成
- 室友新增时会更直观

## 22. 图片怎么用

现在 `位置` 和 `区域` 都可以上传图片。

建议这样用：

1. 给每个收纳盒、抽屉、架子拍一张照片。
2. 在对应的位置或区域编辑里上传图片。
3. 保存后，列表页会直接显示缩略图。
